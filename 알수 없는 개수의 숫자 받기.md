# 알수 없는 개수의 숫자 받기


```cpp

#include <iostream>

int main()
{
	char str[40];
	int n;
	char* str2 = NULL;
	scanf_s("%d\n", &n);
	std::vector<int>* array = new std::vector<int>[n];
	for (int k = 0; k < n; k++)
	{
		gets_s(str, 40);
		char* temp = strtok_s(str, " ", &str2); //공백을 기준으로 문자열 자르기
		while (temp != NULL) { //널이 아닐때까지 반복
			array[k].push_back(atoi(temp));		//문자열 int형으로
			temp = strtok_s(NULL, " ", &str2);	//널문자를 기준으로 다시 자르기
	}
}
	
```
strtok_s를 통해 구분자를 기준으로 문자열 자르기
c스타일의 문자열 char*는 atoi를 통해 int로 변경 (float 형 atof...)
string 클래스 문자열은 stoi를 통해 int로 변경 (float 형 stof...)
