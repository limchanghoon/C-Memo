<h1 id="알수-없는-개수의-숫자-받기">알수 없는 개수의 숫자 받기</h1>
<pre><code>#include &lt;iostream&gt;

int main()
{
	char str[40];
	int n;
	char* str2 = NULL;
	scanf_s("%d\n", &amp;n);

	std::vector&lt;int&gt;* array = new std::vector&lt;int&gt;[n];
	for (int k = 0; k &lt; n; k++)
	{
		gets_s(str, 40);
		char* temp = strtok_s(str, " ", &amp;str2); //공백을 기준으로 문자열 자르기

		while (temp != NULL) { //널이 아닐때까지 반복
			array[k].push_back(atoi(temp));		//문자열 int형으로
			temp = strtok_s(NULL, " ", &amp;str2);	//널문자를 기준으로 다시 자르기
		}
	}
</code></pre>
<p>strtok_s를 통해 구분자를 기준으로 문자열 자르기<br>
c스타일의 문자열 char*는 atoi를 통해 int로 변경 (float 형 atof…)<br>
string 클래스 문자열은 stoi를 통해 int로 변경 (float 형 stof…)</p>
